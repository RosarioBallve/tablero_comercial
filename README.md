## Cómo actualizar los datos
1. Abrí Claude (claude.ai)
2. Subí el Excel `Informe.xlsx` + el archivo `tablero_agro.html`
3. Escribí: "Actualizá el tablero con este Excel"
4. Descargá el HTML nuevo y subilo acá reemplazando el anterior
5. Esperá 1-2 minutos y recargá el link de arriba
